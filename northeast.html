COMPREHENSIVE HTML MASTERY EXAM

1. Complete HTML5 document structure includes:

DOCTYPE declaration
Language attribute
Character encoding
Viewport meta tag

2. Which creates better semantic structure?

<div id="header">
<header>

3. For accessibility, images need:

src attribute only
alt attribute
width and height

4. Best practice for form labels:

<label>Name</label><input>
<label for="name">Name</label><input id="name">

5. Table headers should use:

<td>
<th>
<header>

6. Which link opens in new window securely?

<a href="site.com" target="_blank">
<a href="site.com" target="_blank" rel="noopener">

7. Create collapsible content:

<details><summary></details>
<collapse></collapse>

8. Progress bar element is:

<progress value="50" max="100">
<bar value="50">

9. Main content area uses:

<section>
<main>
<content>

10. Email input type:

<input type="email">
<input type="mail">
